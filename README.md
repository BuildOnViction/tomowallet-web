# TomoWallet - Secure Wallet for TomoChain
TomoWallet web version can send TOMO, TRC20, TRC21. It can help you manage your balance and transactions also.

Support both **TomoChain Mainet** and and **TomoChain testnet**

## Requirements
Node v8.9.0

## Install
`npm install`

## Run
`npm start`

## Build
`npm run build`
